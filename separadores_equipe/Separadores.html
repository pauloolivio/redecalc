Programas para Separação de Equipes

1. Separar alunos por pontuação

Este programa serve para separar as equipes com base na pontuação de cada alunos a partir de um arquivo "csv".

2. Separar aluno em equipes

Este programa serve para separar as equipes de maneira aleatória, com o tamanho da equipe informada pelo docente a partir de um arquivo "csv".

3. Roleta da Sorte

Esse programa roda a rolata para selecionar o aluno participante da atividade.
